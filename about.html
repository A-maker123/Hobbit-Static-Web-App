About This Site

A static character gallery inspired by The Hobbit and Tolkien’s world.

Why I chose this theme

I chose The Hobbit because the characters feel timeless and the story is easy to connect with. Bilbo’s journey is basically a regular person being forced to level up, which makes the character details fun to build and compare.

Design choices

  • I used Bootstrap to keep the layout clean and consistent across pages.
  • The home page uses a grid of cards to make characters easy to scan.
  • The search bar filters characters quickly without needing any backend.
  • The navbar stays the same on every page to meet the usability requirement.

About the project

This website is a static prototype. It demonstrates page navigation, responsive layout, consistent styling, and a simple front end search feature.
